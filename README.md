# Parfait

> Tips: 这是一个使用 vue3 搭配 Arco Design 实现的一个简易后台管理系统模块，可以被用来当作是项目的脚手架。页面风格清新，可以自由配置。

## 功能点
* [x] 垂直、水平布局
* [x] 页面主题配置，包括『界面内容的显示』、『主题色更换』、『多页签配置』、『页面动画』等
* [x] 系统管理模块，包括『用户管理』、『角色管理』等
* [x] 个人设置模块，包括『个人资料修改』、『修改密码』等

## 如何启动？
```TypeScript
pnpm install
pnpm run dev
pnpm run build
```
